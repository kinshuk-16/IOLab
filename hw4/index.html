To-Do list

Completed list

This a list feature. You can add items to the "To-Do" list by writing in the item description in the text box and pressing "Add".


once you have completed the item, you can mark it done by clicking on "Mark Done" button next to individual item. The item will dissapear from the To-Do list and appear in the Completed list.


After marking it done if you feel that item want really quite done, you can even mark it not done! Just click on "Not Done" button next to the item in completed list


Have fun!
